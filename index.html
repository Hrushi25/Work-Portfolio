Credit Card Fraud Detection System

Project Overview

Developed a comprehensive Credit Card Fraud Detection System using machine learning to identify fraudulent transactions with high accuracy. The system leverages transaction patterns, temporal data, and geospatial information to detect potentially fraudulent activities and help financial institutions prevent losses.

Project Screenshots

Key Technologies

  • Languages & Libraries: Python, scikit-learn, XGBoost, Pandas, NumPy
  • Data Visualization: Matplotlib, Seaborn, Recharts
  • Frontend: React, Tailwind CSS
  • Version Control: Git, GitHub

Project Features

  • Machine Learning Pipeline: Implemented robust data preprocessing, trained multiple models, achieved 0.67 AUC-ROC.
  • Data Analysis: Extracted time-based features, calculated geographic distances, identified key predictive features.
  • Interactive Dashboard: Built visualizations for fraud patterns, model performance, and a fraud risk simulator.
  • Performance Metrics: Balanced precision-recall, enabled real-time fraud risk scoring across 500,000+ transactions.

Challenges & Solutions

Handled severe class imbalance (0.39% fraud rate) using sampling techniques and cost-sensitive models. Focused on feature importance analysis to optimize recall without significantly increasing false positives.

Impact & Applications

This system can integrate into real-world payment workflows to prevent financial losses, reduce manual reviews, and provide explainable fraud risk scoring to analysts and decision-makers.

Object Detection System for Self-Driving Cars

Project Overview

Developed a high-performance object detection system for autonomous vehicles using the YOLO (You Only Look Once) architecture. The system accurately identifies and tracks various road objects including vehicles, pedestrians, traffic signs, and potential obstacles in real-time video streams.

Project Demo Video

Technical Details

  • Core Technology: Implemented YOLOv3 algorithm optimized for real-time detection with 90% accuracy.
  • Performance: Achieved 40% reduction in per-frame processing time compared to standard implementations.
  • Detection Capabilities: Recognizes and classifies multiple object categories (cars, trucks, pedestrians, bicycles, traffic lights, etc.).
  • Visualization: Created enhanced UI rendering detection boxes, confidence scores, and object counts.
  • Analytics: Integrated FPS tracking and detection statistics monitoring.

Key Features

  • Real-time detection with color-coded bounding boxes for different object classes.
  • Confidence score display for each detected object.
  • Live performance metrics (FPS counter, detection counts).
  • Adaptive processing for varying lighting and environmental conditions.
  • Comprehensive reporting system for object detection analysis.

Implementation

The system processes video feeds from dashcams or vehicle-mounted cameras, applying computer vision techniques to detect and classify objects. Each detected object is outlined with a bounding box, labeled with its class and confidence score. The processed video maintains high frame rates even in complex traffic scenarios, demonstrating the system's efficiency.

Impact

This object detection system represents a fundamental component for autonomous vehicle perception, enhancing situational awareness and contributing to safer navigation in diverse driving environments.

Technologies Used

Python OpenCV YOLO Computer Vision TensorFlow

Project Overview

It combines facial tension detection with personalized relaxation exercises, providing users with immediate feedback and a fun, engaging way to manage stress through 468 facial landmarks mapping and personalized biofeedback.

Technologies Used

React.js TypeScript MediaPipe FaceMesh Node.js Firebase Convolutional Neural Networks (CNN) FER2013 Dataset

Key Features

  • AI-Powered Facial Detection: Real-time tracking of 468 facial landmarks to assess stress-related muscle tension.
  • Gamified Stress Relief: Interactive mini-games with adaptive difficulty based on user's facial stress levels.
  • Personalized Biofeedback: Visual feedback on facial movements and tension patterns to encourage muscle relaxation.
  • Data Analytics & Tracking: Storage of engagement metrics and stress scores in Firebase for continuous UX optimization.

Impact

FaceVibe successfully merges facial wellness with mental health technology, providing a personalized and engaging solution for stress management. It outperforms traditional wellness apps by significantly improving user adherence to relaxation practices through AI-driven gamification.

Performance Evaluation

  • Model Accuracy: Achieved a test accuracy of 75.90% on the FER2013 dataset for effective facial tension detection.
  • User Engagement: Gamified approach led to high retention rates and consistent participation in facial relaxation exercises.

Future Developments

  • Wearable integration for comprehensive biometric stress tracking.

Predicting Patient Mortality

Project Overview

This project focuses on predicting patient mortality using machine learning techniques, leveraging patient health data to assess the likelihood of mortality based on various clinical features. The goal is to assist healthcare professionals in identifying high-risk patients and providing timely interventions.

Technologies Used

Python Pandas Scikit-learn Matplotlib Seaborn Jupyter Notebook Logistic Regression Random Forest XGBoost

Data Source

The dataset includes a wide array of patient health indicators, such as age, gender, medical history, lab test results, and vital signs. The data was processed and cleaned to ensure high quality for model training and evaluation.

Key Features

  • Data Preprocessing: Performed data cleaning, handled missing values, addressed outliers, and scaled numerical features for model compatibility.
  • Feature Engineering: Created new features to improve the predictive power of the models.
  • Model Development: Built and compared multiple models including Logistic Regression, Random Forest, and XGBoost to predict patient mortality.
  • Model Evaluation: Assessed model performance using accuracy, precision, recall, and AUC-ROC scores to select the best-performing approach.
  • Insights: Identified key features influencing patient mortality, providing actionable insights for healthcare professionals.

Impact

The model enables healthcare providers to proactively identify high-risk patients, enhancing patient care and enabling timely medical interventions. This project showcases the importance of data-driven decision-making in critical healthcare environments and highlights the potential life-saving impact of predictive analytics.
