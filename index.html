Credit Card Fraud Detection System

Project Overview

Developed a comprehensive Credit Card Fraud Detection System using machine learning to identify fraudulent transactions with high accuracy. The system leverages transaction patterns, temporal data, and geospatial information to detect potentially fraudulent activities and help financial institutions prevent losses.

Project Screenshots

Key Technologies

  • Languages & Libraries: Python, scikit-learn, XGBoost, Pandas, NumPy
  • Data Visualization: Matplotlib, Seaborn, Recharts
  • Frontend: React, Tailwind CSS
  • Version Control: Git, GitHub

Project Features

  • Machine Learning Pipeline: Implemented robust data preprocessing, trained multiple models, achieved 0.67 AUC-ROC.
  • Data Analysis: Extracted time-based features, calculated geographic distances, identified key predictive features.
  • Interactive Dashboard: Built visualizations for fraud patterns, model performance, and a fraud risk simulator.
  • Performance Metrics: Balanced precision-recall, enabled real-time fraud risk scoring across 500,000+ transactions.

Challenges & Solutions

Handled severe class imbalance (0.39% fraud rate) using sampling techniques and cost-sensitive models. Focused on feature importance analysis to optimize recall without significantly increasing false positives.

Impact & Applications

This system can integrate into real-world payment workflows to prevent financial losses, reduce manual reviews, and provide explainable fraud risk scoring to analysts and decision-makers.

Predicting Patient Mortality

Project Overview

This project focuses on predicting patient mortality using machine learning techniques, leveraging patient health data to assess the likelihood of mortality based on various clinical features. The goal is to assist healthcare professionals in identifying high-risk patients and providing timely interventions.

Technologies Used

Python Pandas Scikit-learn Matplotlib Seaborn Jupyter Notebook Logistic Regression Random Forest XGBoost

Data Source

The dataset includes a wide array of patient health indicators, such as age, gender, medical history, lab test results, and vital signs. The data was processed and cleaned to ensure high quality for model training and evaluation.

Key Features

  • Data Preprocessing: Performed data cleaning, handled missing values, addressed outliers, and scaled numerical features for model compatibility.
  • Feature Engineering: Created new features to improve the predictive power of the models.
  • Model Development: Built and compared multiple models including Logistic Regression, Random Forest, and XGBoost to predict patient mortality.
  • Model Evaluation: Assessed model performance using accuracy, precision, recall, and AUC-ROC scores to select the best-performing approach.
  • Insights: Identified key features influencing patient mortality, providing actionable insights for healthcare professionals.

Impact

The model enables healthcare providers to proactively identify high-risk patients, enhancing patient care and enabling timely medical interventions. This project showcases the importance of data-driven decision-making in critical healthcare environments and highlights the potential life-saving impact of predictive analytics.
